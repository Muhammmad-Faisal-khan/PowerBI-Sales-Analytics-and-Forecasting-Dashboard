# Sales Analytics & Forecasting Dashboard (Power BI)
Overview
This project presents a complete Sales Analytics Dashboard built in Power BI, providing comprehensive visibility into sales performance, customer behavior, product trends, and future revenue predictions.
It includes two interactive dashboards:

Page 1: Executive Sales Overview
Page 2: Advanced Insights & Forecasting

The solution uses DAX measures, data modeling, date intelligence, and AI-enabled forecasting, making it suitable for decision-makers and demonstrating professional BI skills.
